# GOATED × Dorby — Audit Report

An interactive operations audit and opportunity portfolio prepared by **GOATED** for **Dorby**, from a two-day on-site audit (8 interviews across 7 departments).

**[▶ View the report](https://vanshsfront.github.io/dorby/)** *(enable GitHub Pages to activate this link)*

## What's inside
- Nine ranked automation opportunities, each with problem → solution flowcharts and sourced hours-saved figures
- A self-paced, voice-guided walkthrough that steps through every workflow (auto-play, with an on-screen pause and Enter-to-advance)
- An embedded 2× product demo in the Sales Intelligence section
- Light/dark themes and print-friendly export

`index.html` is fully self-contained — fonts, audio narration, logos, and the demo video are all embedded, so it runs offline with no external dependencies.
